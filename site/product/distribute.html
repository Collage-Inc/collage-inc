Portals

Branded portals for every audience.

Create public-facing portals for any audience — partners, press, franchisees, or your own team. Customize the look, control access, and let people self-serve the assets they need.

  • Custom branding — your logo, colors, and domain
  • Granular access control per portal and per audience
  • Built-in search and filtering for visitors
  • Download tracking and analytics
  • Auto-sync — update the source, the portal updates too
Share Links

Share anything, with anyone.

Generate share links from folders, collages, or individual assets. Add password protection, set expiration dates, and track who's downloading what.

  • One-click link generation for any asset or collection
  • Optional password protection and expiry dates
  • Download tracking — see who accessed what
  • Revoke access instantly by disabling the link
Content Delivery Network

Embed once. Update everywhere.

Embed asset galleries into websites, wikis, internal tools, or anywhere that supports HTML. Content stays in sync — update once, see it everywhere.

  • Copy-paste embed code for any collection or folder
  • Responsive grid that adapts to any container
  • Real-time sync — gallery updates when source changes
  • Customizable layout and display options
